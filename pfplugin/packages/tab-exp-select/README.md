# 分页导航下拉

该插件基于分页导航（DRTAB）的增强，主要是用于将分页导航的分页以下拉形式展现，配置关系界面组后，值变化时加载配置的实体关系界面。**该插件隶属于自定义部件绘制插件（基于分页导航进行扩展）**


## 示例展示

![image](./public/assets/images/scene.png)


## 功能说明

- 支持以下拉的形式展现分页导航的分页项

- 特有的Ui样式
