## 视图界面逻辑
* `onViewMounted`
```javascript
const searchBar = view.getController('searchbar');
if (searchBar) {
searchBar.state.query = view.params.srfquery ? view.params.srfquery : '';
}
```
* `onCreated`
```javascript
view.ctx.evt.on('onRegister', (name, c) => {
    if(name==='tabexppanel'){
        c.state.expViewParams={query: viewParam.srfquery}
    }
});
```
