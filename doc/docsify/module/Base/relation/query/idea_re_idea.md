### 查询条件

(`PRINCIPAL_ID(关联主体标识)` EQ `网页请求上下文.principal_id` AND `PRINCIPAL_TYPE(关联主体类型)` EQ `'idea'` AND `TARGET_TYPE(关联目标类型)` EQ `'idea'`)



### 查询连接
* **IDEA相关N:1（INNER JOIN）DERCUSTOM_RELATION_TARGET_IDEA**<br>
连接关系：[DERCUSTOM_RELATION_TARGET_IDEA](der/DERCUSTOM_RELATION_TARGET_IDEA)<br>
连接实体：[需求](module/ProdMgmt/idea)<br>
连接条件：(`IS_DELETED(是否已删除)` EQ `'0'`)<br>

```sql
SELECT
t1.`CREATE_MAN`,
t1.`CREATE_TIME`,
t1.`CUR_VERSION_ID`,
t1.`ID`,
t1.`LEVEL`,
t1.`NAME`,
t1.`OWNER_TYPE`,
t1.`PARENT_VERSION_ID`,
t1.`PRINCIPAL_ID`,
t1.`PRINCIPAL_TYPE`,
t1.`RELATION_TYPE`,
t1.`TARGET_ID`,
t1.`TARGET_TYPE`,
t1.`TARGET_VERSION_ID`,
t1.`UPDATE_MAN`,
t1.`UPDATE_TIME`
FROM `RELATION` t1 
LEFT JOIN `IDEA` t11 ON t1.`TARGET_ID` = t11.`ID` 


WHERE ( t11.`IS_DELETED` = 0 ) AND ( t1.`PRINCIPAL_ID` = #{ctx.webcontext.principal_id}  AND  t1.`PRINCIPAL_TYPE` = 'idea'  AND  t1.`TARGET_TYPE` = 'idea' )
```
